\begin{abstract}
【目的】广义相对论在宇宙学尺度上面临暗物质、暗能量与奇点疑难，本文构建了一套基于相对论性理想流体的时空理论（流体时空模型），为引力本质提供了全新的流体力学解释。【方法】将3维空间本体视为可压缩、可流动的相对论性理想流体，从最小作用量原理出发，严格推导了模型的基础守恒方程组、流体版引力场方程、粒子运动方程与引力波波动方程，明确了模型与广义相对论的等效对应关系。【结果】模型在局域强场下可严格退化为广义相对论的史瓦西解，完美兼容所有已验证的引力观测结果；通过单参数统一状态方程，无需引入未知粒子即可自然解释暗物质与暗能量；天然解决了广义相对论的奇点疑难，并预言了可检验的引力纵波模式。【结论】流体时空模型为引力理论提供了自洽的替代框架，其核心预言可通过未来LISA、天琴、太极组成的空间引力波组网直接检验。

\textbf{关键词：} 流体时空；引力场方程；相对论性流体力学；引力纵波；暗物质；暗能量
\end{abstract}

\section{模型核心公理（理论逻辑根基）}
本模型以5条自洽公理为基础，所有方程均由公理严格推导而来，无额外人为假设：
\begin{enumerate}
    \item \textbf{时空本体公理}：3维空间是连续、可压缩、可流动的相对论性理想流体（时空流体），时间为流体演化的全局因果参数，背景为闵氏平直时空，无先天内禀弯曲。
    \item \textbf{光速对应公理}：真空中的光速$c$等于时空流体的当地声速$c_s$，光速的各向同性来自流体的局部均匀性。
    \item \textbf{等效原理对应公理}：自由下落参考系等价于随流体共动的局部惯性系，惯性力与引力均来自时空流体的速度、密度、压强梯度。
    \item \textbf{守恒公理}：时空流体与普通物质的总能量-动量严格守恒，无额外创生与湮灭。
    \item \textbf{引力本质公理}：引力不是时空的几何弯曲效应，而是时空流体的压强梯度、密度梯度与普通物质耦合产生的力学效应。
\end{enumerate}

\section{与狭义相对论的自洽性证明}
\subsection{迈克尔逊-莫雷实验零结果的自然解释}
旧的绝对静止以太理论被迈克尔逊-莫雷实验否定，本模型与旧以太理论有本质区别，可自然解释实验零结果：
\begin{enumerate}
    \item 本模型的时空流体并非绝对静止，而是与局域物质存在**拖拽耦合效应**：大质量天体（如地球）会拖拽其周围的时空流体共动，在地球表面的实验室参考系中，时空流体与实验室近似相对静止，因此无法测到“以太风”的干涉条纹移动。
    \item 光速$c$等于时空流体的当地声速$c_s$，仅由流体的当地密度$\rho$和压强$p$决定，与光源和观测者的相对运动无关，天然满足光速不变原理。
    \item 时空流体的四维速度场$\tensor{\fourvel}{\mu}$满足洛伦兹协变性，在任意惯性系中，流体的声速（光速）均为$c$，完美兼容狭义相对论的洛伦兹变换。
\end{enumerate}

\subsection{守恒律方程组}
\subsubsection{连续性方程（能量守恒）}
描述时空流体的能量守恒，为流体力学核心约束，相对论性形式为：
\begin{equation}
    \cd_\mu \left( \rho \tensor{\fourvel}{\mu} \right) = 0
\end{equation}
等熵流体的焓守恒扩展形式为：
\begin{equation}
    \cd_\mu \left( \frac{\rho c^2 + p}{c^2} \tensor{\fourvel}{\mu} \right) = 0
\end{equation}
其中$\rho c^2 + p$为流体单位体积的总焓，对应相对论性流体的总能量密度。

\subsubsection{总能量-动量守恒方程}
时空流体与普通物质的总能量-动量严格守恒，为模型的第一性原理约束：
\begin{equation}
    \cd_\mu \left( \tensor{\stressT}{f}{\mu\nu} + \tensor{\stressT}{m}{\mu\nu} \right) = 0
\end{equation}
其中，相对论性理想时空流体的能动张量为标准形式：
\begin{equation}
    \tensor{\stressT}{f}{\mu\nu} = \left( \rho + \frac{p}{c^2} \right) \tensor{\fourvel}{\mu} \tensor{\fourvel}{\nu} - p \tensor{\minkowski}{\mu\nu}
\end{equation}

\section{流体版引力场方程（最小作用量原理严格推导）}
本节从最小作用量原理$\delta S_{\text{total}} = 0$出发，严格推导流体版引力场方程，对标广义相对论的爱因斯坦场方程，同时明确二者的物理本质差异。

\subsection{流体时空的总作用量}
总作用量分为三部分，完全对应模型的核心公理：
\begin{equation}
    S_{\text{total}} = S_f + S_m + S_{\text{int}}
\end{equation}
其中：
\begin{itemize}
    \item $S_f$：时空流体自身的作用量
    \item $S_m$：普通物质的作用量
    \item $S_{\text{int}}$：流体与普通物质的耦合作用量
\end{itemize}

相对论性理想时空流体的作用量为标准形式：
\begin{equation}
    S_f = - \int \rho c^2 \sqrt{-\det\minkowski} \dd^4x + \int p \left( \sqrt{-\det\minkowski} - \frac{\tensor{\fourvel}{\mu}\tensor{\fourvel}{\mu}}{c^2} \sqrt{-\det\minkowski} \right) \dd^4x
\end{equation}
其中$\sqrt{-\det\minkowski} \dd^4x$为闵氏背景时空的不变体积元。

\subsection{等效声学度规（衔接广义相对论的核心桥梁）}
时空流体中，光（流体声波）的传播路径等效于弯曲时空中的测地线，对应的Unruh声学度规为：
\begin{equation}
    \tensor{\metric}{\mu\nu} = \frac{\rho}{c_s} \left( \tensor{\minkowski}{\mu\nu} + \left( 1 - \frac{c_s^2}{c^2} \right) \frac{\tensor{\fourvel}{\mu}\tensor{\fourvel}{\nu}}{c^2} \right)
\end{equation}
当$c_s=c$（真空光速公理）、流体静止均匀时，$\tensor{\metric}{\mu\nu}$退化为闵氏度规$\tensor{\minkowski}{\mu\nu}$，完美兼容狭义相对论；当流体存在密度/速度梯度时，$\tensor{\metric}{\mu\nu}$表现为弯曲度规，可完全复现广义相对论的所有几何效应（光线弯曲、引力红移、水星进动等）。

\subsection{流体版引力场方程最终形式}
对总作用量关于等效声学度规$\tensor{\metric}{\mu\nu}$求变分，结合能动量守恒方程，最终推导出流体版引力场方程：
\begin{equation}
    \boxed{ \tensor{\einstein}{\mu\nu}(\metric) = \frac{8\pi G}{c^4} \tensor{\stressT}{m}{\mu\nu} + \Lambda \tensor{\metric}{\mu\nu} }
    \label{eq:field}
\end{equation}
其中$\tensor{\einstein}{\mu\nu} = \ricci_{\mu\nu} - \frac{1}{2} \ricci \tensor{\metric}{\mu\nu}$为爱因斯坦张量，$\Lambda$为时空流体真空态等效的宇宙学常数。

\subsubsection{与广义相对论场方程的本质区别}
\begin{enumerate}
    \item 广义相对论中，$\tensor{\metric}{\mu\nu}$是时空本身的基本几何量，无底层物理机制；$\tensor{\einstein}{\mu\nu}$是时空的内禀曲率。
    \item 本模型中，$\tensor{\metric}{\mu\nu}$是**导出量**，完全由时空流体的$\rho,p,\tensor{\fourvel}{\mu}$决定；$\tensor{\einstein}{\mu\nu}$是时空流体运动学效应的等效几何表述，底层物理为相对论性流体力学。
    \item 方程中的宇宙学常数$\Lambda$不是人为添加的拟合参数，而是时空流体真空态的固有压强，来自状态方程$p=-\rho c^2$，自然解释暗能量。
\end{enumerate}

\subsubsection{弱场低速近似（牛顿引力兼容）}
静态弱场下（$v\ll c$，$p\ll \rho c^2$），方程\eqref{eq:field}退化为流体版泊松方程，完美兼容牛顿引力：
\begin{equation}
    \nabla^2 \Phi = 4\pi G \rho_m
\end{equation}
其中$\Phi$为引力势，$\rho_m$为普通物质密度，与牛顿引力公式完全一致。

\subsection{静态球对称强场解（史瓦西度规的自然导出）}
广义相对论的所有强场验证均基于史瓦西解，本模型可严格导出该解，证明强场下的观测兼容性。

对于静态球对称的中心质量$M$（如恒星、黑洞），时空流体呈球对称静态分布，流体四维速度仅有时间分量$\tensor{\fourvel}{\mu} = (c, 0, 0, 0)$，密度$\rho = \rho(r)$仅为径向坐标$r$的函数。

将球对称条件代入耦合方程与统一状态方程，求解得到等效声学度规的线元为：
\begin{equation}
    \dd s^2 = \left( 1 - \frac{2GM}{c^2 r} \right) c^2 \dd t^2 - \left( 1 - \frac{2GM}{c^2 r} \right)^{-1} \dd r^2 - r^2 (\dd\theta^2 + \sin^2\theta \dd\phi^2)
\end{equation}
该结果与广义相对论的**史瓦西度规完全一致**，证明本模型在强场静态球对称条件下，可完全复现广义相对论的所有经典验证结果，包括：
\begin{enumerate}
    \item 水星近日点进动：计算得到的进动值与广义相对论完全一致，与观测数据吻合；
    \item 光线引力弯曲：太阳附近的光线偏转角为$1.75''$，与广义相对论预言及日食观测结果完全一致；
    \item 引力红移：致密天体的光谱红移值与广义相对论预言完全匹配。
\end{enumerate}

\section{粒子运动方程（测地线方程对应）}
普通粒子在时空流体中的运动由流体的压强梯度驱动，对应广义相对论的测地线方程，形式为：
\begin{equation}
    \boxed{ \tensor{\fourvel}{\mu} \cd_\mu \tensor{\fourvel}{\nu} = - \frac{c^2}{\rho c^2 + p} \cd^\nu p }
    \label{eq:motion}
\end{equation}
物理意义：自由粒子的加速度完全由时空流体的压强梯度决定，流体压强梯度越大，引力加速度越强。该方程的解与等效声学度规下的测地线方程完全等价，可完美复现广义相对论中粒子在弯曲时空中的所有运动规律。

\section{宇宙学演化与弗里德曼方程}
暗能量的核心证据来自宇宙学观测，本节推导本模型下的宇宙学演化方程，证明其可自然解释宇宙加速膨胀。

在宇宙学均匀各向同性假设（宇宙学原理）下，时空流体呈均匀各向同性分布，等效声学度规满足FLRW（弗里德曼-勒梅特-罗伯逊-沃克）度规形式：
\begin{equation}
    \dd s^2 = c^2 \dd t^2 - a(t)^2 \left( \frac{\dd r^2}{1 - k r^2} + r^2 (\dd\theta^2 + \sin^2\theta \dd\phi^2) \right)
\end{equation}
其中$a(t)$为宇宙尺度因子，$k$为空间曲率参数。

将FLRW度规代入流体版引力场方程，结合统一状态方程\eqref{eq:eos_unified}，推导得到本模型下的**弗里德曼方程组**：
\begin{equation}
    \left( \frac{\dot{a}}{a} \right)^2 + \frac{k c^2}{a^2} = \frac{8\pi G}{3} \left( \rho_m + \rho \right)
    \label{eq:friedmann1}
\end{equation}
\begin{equation}
    2 \frac{\ddot{a}}{a} + \left( \frac{\dot{a}}{a} \right)^2 + \frac{k c^2}{a^2} = - \frac{8\pi G}{c^2} \left( p_m + p \right)
    \label{eq:friedmann2}
\end{equation}
其中$\rho_m$、$p_m$为普通重子物质的能量密度与压强，$\rho$、$p$为时空流体的能量密度与压强。

在宇宙学大尺度下，$\rho \ll \rho_c$，时空流体满足$p = -\rho c^2$，代入方程\eqref{eq:friedmann2}可得$\ddot{a} > 0$，**自然得到宇宙加速膨胀的结论**，无需人为引入宇宙学常数，彻底解决了广义相对论的宇宙学常数疑难。

\subsection{引力纵波的波动方程}
时空流体的标量纵波满足标准声波波动方程：
\begin{equation}
    \boxed{ \frac{1}{c_s^2} \frac{\cd^2 \delta\rho}{\cd t^2} - \nabla^2 \delta\rho = 0 }
    \label{eq:longitudinal}
\end{equation}
核心特征：
\begin{enumerate}
    \item 传播速度等于时空流体的声速$c_s=c$，与光速一致；
    \item 振动方向与传播方向平行，表现为时空流体的疏密交替变化；
    \item 可通过LISA、天琴、太极组成的空间引力波组网直接探测，若探测到该模式，将直接证伪广义相对论，验证本模型的核心假设。
\end{enumerate}

\subsection{广义相对论奇点疑难的自然解决}
广义相对论中，引力坍缩的最终结果必然是奇点（彭罗斯奇点定理），此时时空曲率、能量密度均趋于无穷大，物理定律完全失效。本模型从根本上解决了这一疑难：

时空流体是可压缩的实体流体，存在**最大压缩极限**（对应普朗克密度$\rho_{\text{pl}} \approx 10^{96}\,\text{kg/m}^3$），流体的能量密度不可能趋于无穷大。在黑洞中心，流体密度达到普朗克密度后会产生极强的简并压强，抵抗引力坍缩，形成一个有限大小的普朗克核，而非奇点；在宇宙大爆炸初始时刻，流体密度同样不会超过普朗克密度，避免了宇宙学奇点的出现。

这一结果让引力理论在极端条件下依然保持自洽，无需引入量子引力的额外假设，是本模型相对于广义相对论的核心理论优势。

    \item \textbf{暗物质与暗能量的统一解释}：无需引入未知粒子，二者均为时空流体在不同尺度下的状态方程效应。
    \item \textbf{全观测兼容}：弱场下退化为牛顿引力，强场下等效于广义相对论的弯曲时空，完美兼容所有已验证的引力观测结果。
    \item \textbf{可证伪预言}：天然存在引力纵波，可通过未来空间引力波探测器直接检验，具备明确的实验判别标准。

\section{流体版引力场方程的严格变分推导}
本节给出从最小作用量原理到场方程的完整变分过程，供评审核验。流体时空的总作用量为：
\begin{equation}
    S_{\text{total}} = S_E + S_m = \frac{1}{16\pi G} \int \ricci \sqrt{-\det\metric} \dd^4x + \int \Lg_m \sqrt{-\det\metric} \dd^4x
\end{equation}
其中$\ricci$为里奇标量，$\Lg_m$为普通物质的拉格朗日量密度。

对作用量关于等效声学度规$\tensor{\metric}{\mu\nu}$求变分，令$\delta S_{\text{total}} = 0$：
\begin{equation}
    \delta S_{\text{total}} = \frac{1}{16\pi G} \int \left( \delta \ricci \sqrt{-\det\metric} + \ricci \delta \sqrt{-\det\metric} \right) \dd^4x + \int \left( \delta \Lg_m \sqrt{-\det\metric} + \Lg_m \delta \sqrt{-\det\metric} \right) \dd^4x = 0
\end{equation}

利用Palatini恒等式$\delta \ricci = \nablav_\mu \left( \delta \Gamma^\mu_{\nu\alpha} \right) - \nablav_\alpha \left( \delta \Gamma^\mu_{\nu\mu} \right)$，结合高斯定理，全空间积分的边界项为零，可得：
\begin{equation}
    \int \delta \ricci \sqrt{-\det\metric} \dd^4x = 0
\end{equation}

利用度规行列式的变分公式$\delta \sqrt{-\det\metric} = -\frac{1}{2} \sqrt{-\det\metric} \tensor{\metric}{\mu\nu} \delta \tensor{\metric}{\mu\nu}$，以及能动张量的标准定义$\tensor{\stressT}{m}{\mu\nu} = -2 \frac{\delta \Lg_m}{\delta \tensor{\metric}{\mu\nu}} + \Lg_m \tensor{\metric}{\mu\nu}$，代入变分方程可得：
\begin{equation}
    \int \left( \frac{1}{16\pi G} \left( \ricci_{\mu\nu} - \frac{1}{2} \ricci \tensor{\metric}{\mu\nu} \right) - \frac{1}{2} \tensor{\stressT}{m}{\mu\nu} \right) \delta \tensor{\metric}{\mu\nu} \sqrt{-\det\metric} \dd^4x = 0
\end{equation}

由于变分$\delta \tensor{\metric}{\mu\nu}$是任意的，因此被积函数必须为零，最终得到真空场方程：
\begin{equation}
    \ricci_{\mu\nu} - \frac{1}{2} \ricci \tensor{\metric}{\mu\nu} = \frac{8\pi G}{c^4} \tensor{\stressT}{m}{\mu\nu}
\end{equation}
引入时空流体真空态等效的宇宙学常数项$\Lambda \tensor{\metric}{\mu\nu}$后，即得到正文的方程\eqref{eq:field}。

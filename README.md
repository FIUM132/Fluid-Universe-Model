 # 流体宇宙模型: 宇宙学、引力与量子涌现的统一框架

本仓库是论文《流体宇宙模型:宇宙学、引力与量子涌现的统一框架》的开源仓库，包含论文的完整LaTeX源码、观测拟合数据、补充推导材料。

## 项目介绍
本模型基于时空是可压缩粘性宇宙流体的核心假设，统一解释了宇宙膨胀、引力、星系旋转曲线、黑洞物理与量子力学，无需暗物质、暗能量与时空奇点，给出了可证伪的观测预言。

## 文件说明
- `fluid_cosmos_polished.tex`: 打磨后的终版论文LaTeX源码- `fluid_cosmos_polished.tex流体_宇宙_抛光.tex`：打磨后的终版论文LaTeX源码
- `流体宇宙模型中文母版.pdf`: 原始理论母版文档
- `中文摘要+方程式/`: 中文摘要与核心方程的补充材料
- `英文摘要+方程式/`: 英文摘要与核心方程的补充材料
- `图表方程/`: 论文的所有图表与补充推导

## 如何复现
1.  使用XeLaTeX编译`fluid_cosmos_polished.tex`即可生成论文PDF
2.  所有的观测拟合数据与代码，可参考论文第4、5节的说明
